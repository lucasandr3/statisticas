# Estatísticas

Api que retona estatísticas

### Requisitos

- PHP - 8.3
- Composer

### Instalação

- Clone o repositório
```
git clone git@github.com:lucasandr3/statisticas.git
```
- Instale as dependências do Composer:
```
composer install
```
- Copie o arquivo .env.example para .env:
```
cp .env.example .env
```
- Gere a chave da aplicação:
```
php artisan key:generate
```
